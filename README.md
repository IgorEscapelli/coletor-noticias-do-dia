Este projeto é uma aplicação web construída em Python, Flask e BeautifulSoup, projetada para automatizar a coleta diária de notícias de múltiplas fontes.
O sistema suporta:

Scraping HTML tradicional,

Leitura de feeds RSS,

Consumo de APIs públicas (como a do G1),

Filtragem inteligente por data de publicação,

Consolidação dos resultados em um único arquivo JSON.

A interface web permite que o usuário execute a coleta com um único clique e baixe o arquivo final com facilidade. Foi criado para centralizar notícias regionais, facilitando análises, integrações ou consumo em outros sistemas.
